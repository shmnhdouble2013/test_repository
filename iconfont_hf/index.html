如何使用icon-font ?

如果您开发是web前台页面，并且继承了公用页头，那么就无需再引入iconfont.css或iconfont-https.css了，因为公用页头里已经集中引入！(好房宝业务使用的是https安全加密协议，目前是继承公用页头的基础上覆盖引入iconfont-https.css)反之往下看：

第一步：项目中引入iconfont字体文件 2种方式： 

        方式一：常规方式，现有的php框架自动引入！直接在页面相应的.php资源引入里配置(引用static仓库master分支当前最新的字体文件)，示例代码如下：
            /**
             * 使用的样式
             * @return array
             */
            static function useOStyle(){
                return array(
                    'static://css/iconfont/iconfont.css',
                );
            }

        方式二：手动引入！直接引用线上最新发布的iconfont.css 字体文件(和PHP框架引入是同一来源，static仓库master分支当前最新的字体文件)，比如：
            代码(http普通业务)：  <link href="http://static.pinganfang.com/css/iconfont/iconfont.css?update=141208" rel="stylesheet" type="text/css">

            字体库文件部署已经优化，无需再特殊引入iconfont-https.css文件，已经引入的代码依然有效，可删除可不删。为了维护的统一性，建议后期维护时候删除iconfont-https.css文件，只引入iconfont.css文件！
            代码(https好房宝业务): <link href="https://hfb.pinganfang.com/static/201412151658/css/iconfont/iconfont-https.css" rel="stylesheet" type="text/css">
第二步：给需要使用字体icon的html元素 添加字体class标示 .iconfont 

        <span class="iconfont"></span>
第三步：挑选相应图标并获取字体编码，引用元素上 

        <span class="iconfont">&#xe643;</span>
