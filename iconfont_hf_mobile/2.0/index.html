如何使用icon-font ?

如果您开发是web前台页面，并且继承了公用页头，那么就无需再引入iconfont.css了，因为公用页头里已经集中引入！反之往下看：

第一步：项目中引入iconfont字体文件 2种方式： 

        方式一：常规方式，旧php框架自动引入(马上要废弃)！直接在页面相应的.php资源引入里配置(引用static仓库master分支当前最新的字体文件)，示例代码如下：
            /**
             * 使用的样式
             * @return array
             */
            static function useOStyle(){
                return array(
                    'static://common/iconfont/mobile/iconfont.css'
                );
            }

        方式二：手动直接引入！直接引用线上最新发布的1.0.0版本的iconfont.css 字体文件(你可以根据需求来指定引入已经发布的版本文件，初始化为1.0.0)，比如：
            <link href="http://static.anhouse.com/common/1.0.0/iconfont/mobile/iconfont.css" rel="stylesheet" type="text/css">
第二步：给需要使用字体icon的html元素 添加字体class标示 .iconfont 

        <span class="iconfont"></span>
第三步：挑选相应图标字体编码 或 fontclass 引用元素上(2种方式) 

        1、字体编码方式：
        <span class="iconfont">&#xe643;</span>

        2、icon-font class方式：
        <span class="iconfont icon-back"></span>
第四步：看下效果吧(2种方式效果都一样，根据喜好选择开发方式)
